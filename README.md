# go-pivotaltracker #

Pivotal Tracker API client for Go (Golang)

## Usage ##

The API is rather incomplete, so it's tagged with `v0`:

```go
import "gopkg.in/salsita/go-pivotaltracker/v0/v5/pivotal"
```
